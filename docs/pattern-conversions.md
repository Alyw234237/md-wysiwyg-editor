# Text Pattern Conversions

| Pattern | Conversion |
| --- | --- |
| `# ` | H1 |
| `## ` | H2 |
| `### ` | H3 |
| `#### ` | H4 |
| `##### ` | H5 |
| `###### ` | H6 |
| `*...*` | Italic |
| `_..._` | Italic |
| `**...**` | Bold |
| `__...__` | Bold |
| `***...***`` | Bold and italic |
| `___...___` | Bold and italic |
| `~~Strikethrough~~` | Strikethrough |
| `^...^` | Superscript |
| `* ` | Bulleted list |
| `+ ` | Numbered list |
| `- ` | Numbered list |
| `1. ` | Numbered list |
| `> ` | Block quote |
| `\\`...\\`` | Code |
| `\\`\\`\\` | Code block |
| `--` | En dash |
| `---` | Em dash |
| `===` | Horizontal line |
